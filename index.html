Controls:

Left click - Mark square as Completed (fills with your color)
Right Click - Border-mark a square (outlines with your color)
Scroll Wheel - Cycle: Default > Border-marked > Completed
Hotkeys: 1, 2, 3 - Mark square as Completed (Rush Mode Only)

Icons Toggle - Available on the board page. Toggles between displaying icons or text
Wheel Mode Toggle - Available on the board page. Toggles between using the Scroll Wheel to mark squares or incrementing/decrementing a counter.

Modifiers

- Shiny Goals: If enabled, will set random squares to "Shiny", putting a special border around goals.

Sharing

- Copy Preset Link: Saves all generator options into a URL (with seed if generated). Anyone who opens it sees the same settings.
- Copy Board Link: Available on the board page. Saves the exact board (including seed) for direct sharing.
